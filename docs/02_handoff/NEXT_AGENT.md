## 3. Разрешённый фокус

### Этап 2 — Логика Агента и AI-анализ

Твоя задача — реализовать агентный слой, который:

1. Принимает сырую команду пользователя.
2. Извлекает поисковые и отсекающие критерии.
3. Формирует структурированную команду парсеру.
4. Получает результаты парсинга.
5. Выполняет семантический анализ резюме (soft skills, контекст).
6. Формирует осмысленный результат для пользователя.
