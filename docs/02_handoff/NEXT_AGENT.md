## 3. Разрешённый фокус: пункт 6 (Логика Агента и AI-анализ)

Твоя задача — реализовать агентный слой, который:

1) принимает сырую команду пользователя (естественный язык);
2) извлекает:
   - поисковые критерии,
   - критерии отсечения (must / must-not),
   - критерии семантического анализа (soft skills, контекст);
3) формирует структурированную команду существующему Local MVP (без изменений транспорта/парсеров);
4) получает результаты парсинга;
5) выполняет семантический анализ и ранжирование;
6) формирует осмысленный результат для пользователя (shortlist/оценки/объяснения + ссылки на источник).

## 4. Запрещено

- начинать контейнеризацию/облако;
- рефакторить или усложнять пункты 1–5.

## 5. Цель

Сделать Local MVP управляемым инструментом сбора данных, а агент — единственным носителем “логики и смысла” системы.
